# ResearchAPI
Die JSON Datei Follower beinhaltet die gesammelten Follower der Afd.
Das Sammelskript enthält zum einen das tatsächliche Sammelskript und kommentiert angefügt sind die Anfragen die probiert wurden, aber nicht funktionieren/nur schlecht funktionieren, für spätere Verwendung.
Die datenbank ist einmal als reine Struktur und als kompletter Export abgelegt.
